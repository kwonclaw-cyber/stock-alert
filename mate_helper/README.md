# 메이트포스 매출 자동수집 도우미

월마감 페이지의 **"⤓ 매출 가져오기"** 버튼을 누르면, 이 프로그램이 메이트포스에 대신 로그인해
그 달 매출을 받아 자동으로 채워줍니다. (아이디/비번은 이 폴더의 `.env` 파일에만 저장, 외부 전송 없음)

---

## ✅ 처음에 딱 3단계 (한 번만)

### 1단계 — 파이썬 설치 (한 번만)
- https://www.python.org/downloads/ 접속 → 노란 **Download** 버튼으로 설치
- ⚠️ 설치 화면 **맨 아래 "Add python.exe to PATH" 체크박스를 꼭 체크**하고 설치하세요.

### 2단계 — `설치.bat` 더블클릭
- 이 폴더의 **`설치.bat`** 을 더블클릭하면 필요한 게 자동으로 깔립니다. (처음엔 몇 분 걸려요)
- 중간에 **메모장이 열리면**, 메이트포스 아이디/비번을 넣고 **저장(Ctrl+S) 후 닫기**:
  ```
  MATETECH_USERNAME=밥구정_아이디
  MATETECH_PASSWORD=밥구정_비번
  MATETECH_BRAND=밥구정로데오
  MATETECH_USERNAME_2=육식_아이디
  MATETECH_PASSWORD_2=육식_비번
  MATETECH_BRAND_2=육식사관학교
  ```
  ("= " 오른쪽에만 입력하면 됩니다. 빈칸 없이.)

### 3단계 — 끝!
이제 설치는 끝났습니다. 사용할 때는 아래만 하면 됩니다.

---

## ▶ 매번 쓸 때 (2단계)

1. **`run.bat`** 더블클릭 → 검은 창이 뜨고 켜져 있으면 준비 완료 (이 창은 닫지 말고 켜두세요)
2. 브라우저에서 **http://localhost:8765/월마감.html** 접속
   → 월 선택 → **"⤓ 매출 가져오기"** → 1~2분 뒤 매출 자동 입력 → 정산 계산

> 끝나면 검은 창(run.bat)을 닫으면 종료됩니다.

---

## 참고
- 가져오는 자료 = 메이트포스 **매장별 매출**(월마감의 "매출" 입력).
- **물류 발주원장**은 메이트포스가 아니라 푸드가이드라서 계속 수동 업로드입니다.
- 자격증명은 PC의 `.env` 에만 있고, 모든 처리는 내 PC 안(localhost)에서만 됩니다. `.env` 는 남에게 공유하지 마세요.

## 잘 안 될 때
- "수집 실패": `run.bat` 검은 창이 켜져 있는지 확인 / 도우미 창의 빨간 글씨를 알려주세요.
- 로그인 실패: 메모장으로 `.env` 열어 아이디·비번 다시 확인.
- 파이썬 없음 메시지: 1단계 다시 (PATH 체크 빠뜨렸으면 파이썬 재설치).
