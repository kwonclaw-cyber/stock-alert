# 메이트포스 매출 자동수집 도우미

월마감 페이지의 **"메이트포스에서 가져오기"** 버튼이 호출하는 작은 로컬 프로그램입니다.
이 프로그램이 메이트포스에 로그인해 지정한 월의 매출 엑셀을 받아 페이지로 돌려줍니다.
**자격증명(아이디/비번)은 이 폴더의 `.env` 에만 저장**되며 외부로 전송되지 않습니다.

> 한 줄 요약: 이 프로그램을 켜두면 → 월마감 페이지에서 월 선택 + 버튼 → 매출이 자동으로 채워집니다.

## 처음 1회 설치 (PC)

1. **Python 설치** (3.10+). Windows면 python.org에서 설치 시 "Add to PATH" 체크.
2. 이 `mate_helper` 폴더에서 명령창 열고:
   ```
   pip install -r requirements.txt
   playwright install chromium
   ```
3. `.env.example` 을 복사해 **`.env`** 파일을 만들고 메이트포스 계정을 입력:
   ```
   MATETECH_USERNAME=밥구정아이디
   MATETECH_PASSWORD=밥구정비번
   MATETECH_BRAND=밥구정로데오
   MATETECH_USERNAME_2=육식아이디
   MATETECH_PASSWORD_2=육식비번
   MATETECH_BRAND_2=육식사관학교
   ```

## 매번 사용할 때

1. 프로그램 실행: 폴더의 **`run.bat`(윈도우)** 더블클릭, 또는 명령창에서 `python server.py`
   - "http://localhost:8765 ... 계정 N개 로드" 가 뜨면 준비 완료. **이 창을 켜둔 채로** 사용.
2. **월마감 페이지 열기** — 두 가지 방법 중 하나:
   - (권장) 브라우저에서 **http://localhost:8765/월마감.html** 접속
     → `월마감.html` 파일을 이 `mate_helper` 폴더에 같이 두면 됩니다. (같은 주소라 가장 안정적)
   - 또는 `월마감.html` 을 그냥 더블클릭해서 열기.
3. 페이지에서 **월 선택 → "⤓ 매출 가져오기"** → 1~2분 후 매출이 자동 입력됩니다.
   (물류 발주원장은 메이트포스가 아니라 푸드가이드라서 계속 수동 업로드입니다.)

## 동작 / 보안
- 로그인·다운로드 흐름은 저장소의 `sales_report.py`(이미 운영 중)와 동일한 방식입니다.
- 받아오는 자료: 메이트포스 **기간별 매출분석(매장별)** = 월마감의 매출(순/실/완료주문) 입력.
- 모든 처리는 PC 안(localhost)에서만 이루어집니다. 계정정보는 `.env` 로컬 파일에만 있습니다.
- `.env` 는 절대 저장소에 올리지 마세요(공유 금지).

## 문제 해결
- "수집 실패 — 도우미 실행 확인": `server.py` 가 실행 중인지, 포트(8765)가 맞는지 확인.
- 로그인 실패: `.env` 계정/비번 확인.
- 포트 변경: `.env` 에 `MATE_HELPER_PORT=다른포트` 추가 후, 페이지 `HELPER_URL` 도 맞춰야 함(문의 주세요).
