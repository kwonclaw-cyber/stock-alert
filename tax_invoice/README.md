# 광고지원금 안내문 생성기

본사(산들그린)가 가맹점 사장님께 **광고지원금에 대한 세금계산서/현금영수증을 본사 앞으로 발행해 달라**고
요청하는 안내문을, 원본과 같은 **표 형태**로 매장별로 만들어 카카오톡으로 보내기 위한 정적 웹페이지입니다.
서버 없이 `tax_invoice/index.html`을 브라우저로 열면 동작합니다.

## 사용 흐름 (안내문 만들기 탭)
선택 3가지만 정하면 표 형태 안내문이 실시간으로 완성됩니다.
1. **과세여부** : 세금계산서 / 현금영수증
2. **매장명** : 등록된 매장 선택 또는 직접 입력
3. **금액** :
   - 세금계산서 → **총액(부가세 포함)** 입력 시 공급가액 = 총액/1.1, 세액 = 총액 − 공급가액 자동 계산
   - 현금영수증 → 입력 금액이 공급가액·합계로, 세액은 표기 안 함

완성된 표는 아래 버튼으로 전달합니다.
- **📋 표 복사** : 표를 복사해 카카오톡 등에 붙여넣기 (서식 미지원 앱에서는 텍스트로 붙음)
- **🖼 이미지로 저장** : PNG로 저장 → 카톡에 첨부 (표가 깨지지 않아 권장)
- **🖨 인쇄 / PDF** : 인쇄하거나 PDF로 저장

## 매장 기초정보 탭
- 매장명/대표자/연락처/기본 과세여부/기본 금액/메모 등록·수정·삭제
- 등록 매장은 안내문의 매장 선택 목록에 표시되고, 선택 시 기본 과세여부·금액이 자동 채워집니다.
- **구글 시트로 일괄 등록**: 구글 스프레드시트에서 셀 범위를 복사해 붙여넣거나 CSV를 붙여넣으면 한 번에 등록됩니다.
  - 열 순서: 매장명 · 대표자 · 연락처 · 과세여부 · 금액 · 메모 (첫 줄이 머리글이면 자동 인식)
  - 과세여부: `세금계산서/세금/과세`→세금계산서, `현금영수증/현금`→현금영수증 (비우면 세금계산서)
  - 금액은 콤마가 있어도 인식, "기존 매장 전체 교체" 옵션 제공

## 본사 정보 탭
- 상호명/사업자번호/사업장/업태/종목/이메일/성명(세금계산서·현금영수증)/경영지원실 연락처
- 산들그린 정보로 기본 세팅되어 있으며, 성명은 과세여부에 따라 자동 표기(세금계산서=이용석 / 현금영수증=이용환).

## 데이터·백업
- 매장·본사 정보는 브라우저 `localStorage`에 저장됩니다.
- 다른 PC로 옮기거나 백업하려면 우상단 **백업 내보내기 / 가져오기**(JSON)를 사용하세요.

## 참고: 카카오톡 일괄발송
개인 카카오톡은 자동·일괄 발송이 불가합니다. 본 페이지는 매장별로 안내문을 만들어 **복사/이미지/PDF**로 직접
전달하는 방식입니다. 정식 일괄발송이 필요하면 알림톡·문자 대행사(예: 솔라피) 연동으로 확장할 수 있습니다.
