# 광고지원금 계산서 발행요청 관리 페이지

본사(공급받는자)가 각 가맹점 사장님께 **광고지원금에 대한 세금계산서/현금영수증을 본사 앞으로 발행해 달라**는
안내문을 매장별로 자동 완성해 보내고, 발행 진행 상태를 관리하는 정적 웹페이지입니다.
서버 없이 `tax_invoice/index.html`을 브라우저로 열면 동작합니다.

## 기능
- **매장 기초정보**: 매장명/대표자(사장님)/연락처/기본 유형(세금계산서·현금영수증)/월 지원금/메모
- **월별 발행관리**: 월 선택 → 매장별 유형·금액·상태 관리
  - 금액(합계, 부가세 포함) 입력 시 자동 분리
    - 세금계산서: 공급가액 = 합계/1.1, 세액 = 합계 − 공급가액
    - 현금영수증: 공급가액 = 합계, 세액 없음
  - 상태: 미발송 / 요청완료 / 발행완료 / 미발행, 상태별 집계·필터
- **안내문 발송**: 세금계산서용·현금영수증용 양식 2종(매장 유형에 따라 자동 선택)
  - 변수 자동 치환: `{매장명} {대표자} {월} {공급가액} {세액} {합계} {상호명} {성명} {사업자번호} {사업장} {업태} {종목} {이메일} {경영지원실}`
  - 대상 필터(전체/미발송만/미발행만), 일괄 미리보기
  - **전체 복사**(카카오톡 붙여넣기) / **CSV 내보내기**(알림톡·문자 대행사 업로드용)
- **본사 정보**: 상호명/사업자번호/사업장/업태/종목/이메일/성명(세금계산서·현금영수증)/경영지원실 연락처
  - 산들그린 정보로 기본 세팅되어 있음
- **백업 내보내기/가져오기**(JSON)

데이터는 브라우저 `localStorage`에 저장됩니다. 백업/이전은 **백업 내보내기**를 사용하세요.

## 카카오톡 일괄 발송
- 개인 카카오톡은 자동·일괄 발송이 불가합니다(정책상 금지, 공식 API 없음).
- 정식 일괄발송: **알림톡**(카카오 채널+대행사+템플릿 심사) 또는 **문자 SMS/LMS**(대행사만 있으면 즉시).
- 추천 대행사: **솔라피(Solapi)**. 현재는 **CSV 내보내기** 파일을 대행사에 업로드해 일괄발송하고,
  추후 이 저장소의 파이썬으로 API 자동발송·매월 말 스케줄까지 확장 가능합니다.

## 사용 방법
1. `tax_invoice/index.html`을 브라우저로 엽니다.
2. `본사 정보` 확인/수정 후 저장합니다.
3. `매장 기초정보`에서 매장(사장님·휴대폰·유형·지원금)을 등록합니다.
4. `월별 발행관리`에서 대상 월의 유형·금액·상태를 관리합니다.
5. `안내문 발송`에서 일괄 미리보기 → 전체 복사 또는 CSV 내보내기로 발송합니다.
