# 매장 청구·안내 관리 페이지

매장 기초정보를 등록하고, 매월 청구 상태를 관리하며, 매장별 **안내문(청구/미수 안내)** 을
자동 완성해 복사·CSV로 내보내는 정적 웹페이지입니다.
서버 없이 `tax_invoice/index.html`을 브라우저로 열면 동작합니다.

## 기능
- **매장 기초정보**: 매장명/대표자/연락처(휴대폰)/사업자번호/이메일/월 청구금액/메모 등록·수정·삭제
- **월별 청구관리**: 월 선택 → 매장별 상태 관리
  - 상태: 미발송 / 발송완료 / 입금완료 / 미입금 (상태 클릭으로 변경, 날짜 자동 채움)
  - 상태별 건수·금액 집계, 상태별 필터
- **안내문 발송(작성/미리보기)**:
  - 변수 템플릿으로 본문/제목 작성 → 매장별로 자동 치환
  - 변수: `{매장명} {대표자} {사업자번호} {연락처} {청구금액} {월} {마감일} {상태} {상호} {담당자} {입금계좌} {발신연락처}`
  - 대상 필터(전체/미발송만/미입금만), 일괄 미리보기
  - **전체 복사**(카카오톡 붙여넣기용), **CSV 내보내기**(알림톡·문자 대행사 업로드용)
- **발신 설정**: 상호/담당자/연락처/입금계좌
- **백업 내보내기/가져오기**(JSON)

데이터는 브라우저 `localStorage`에 저장됩니다. 백업/이전은 **백업 내보내기**를 사용하세요.

## 카카오톡 일괄 발송에 대해
- **개인 카카오톡은 프로그램으로 자동·일괄 발송할 수 없습니다**(카카오 정책상 금지, 공식 API 없음).
- 사업용 일괄발송 정식 방법:
  - **알림톡(AlimTalk)**: 카카오 비즈니스 채널 + 발송 대행사 + 템플릿 사전심사 필요.
  - **문자(SMS/LMS)**: 대행사만 있으면 템플릿 심사 없이 바로 발송 가능.
- 추천 대행사: **솔라피(Solapi)** — 무료가입 후 API 키 발급, 문자 즉시 발송 가능.
- 현재는 이 페이지의 **CSV 내보내기**를 대행사 사이트에 업로드해 일괄발송하면 됩니다.
  추후 이 저장소의 파이썬으로 대행사 API 자동발송·스케줄까지 확장할 수 있습니다.

## 사용 방법
1. `tax_invoice/index.html`을 브라우저로 엽니다.
2. `발신 설정`에서 상호·담당자·입금계좌를 저장합니다.
3. `매장 기초정보`에서 매장(휴대폰번호 포함)을 등록합니다.
4. `월별 청구관리`에서 대상 월의 금액·상태를 관리합니다.
5. `안내문 발송`에서 양식을 작성/저장하고, 일괄 미리보기 → 전체 복사 또는 CSV 내보내기로 발송합니다.
