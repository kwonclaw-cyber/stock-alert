# 매장 세금계산서 관리 페이지

매장 기초정보를 등록하고, 매월 세금계산서 발행 상태를 관리하는 정적 웹페이지입니다.
별도 서버 없이 `tax_invoice/index.html` 파일만 브라우저로 열면 동작합니다.

## 기능
- **매장 기초정보**: 매장명/사업자번호/대표자/연락처/이메일/월 청구금액/메모 등록·수정·삭제
- **월별 발행관리**: 월을 선택해 매장별 상태 관리
  - 상태: 미발급 / 발급완료 / 입금완료 / 미지급 (상태 클릭으로 변경, 날짜 자동 채움)
  - 상태별 건수·금액 집계, 상태별 필터
- **계산서 발행(인쇄/PDF)**: 행의 `발행` 버튼 → 세금계산서/계산서 양식을 새 창에 작성 → 인쇄 또는 PDF 저장
  - 종류 선택: **세금계산서**(공급가액+부가세 10%) / **계산서**(면세, 부가세 없음)
- **공급자 설정**: 내 사업자 정보(공급자란에 자동 기입)
- **백업 내보내기/가져오기**: 데이터를 JSON으로 저장·복원

데이터는 브라우저 `localStorage`에 저장됩니다(해당 브라우저에만 보관). 다른 PC로 옮기거나
백업하려면 우상단 **백업 내보내기**를 사용하세요.

## 사용 방법
1. `tax_invoice/index.html`을 브라우저로 엽니다. (GitHub Pages로도 배포 가능)
2. `공급자 설정` 탭에서 내 사업자 정보를 저장합니다.
3. `매장 기초정보` 탭에서 매장을 등록합니다.
4. `월별 발행관리` 탭에서 대상 월을 고르고, 매장별 상태를 관리하거나 `발행`으로 양식을 출력합니다.

## 참고: 국세청 전자(세금)계산서 정식 발행
법적 효력이 있는 **국세청 전자세금계산서** 신고는 홈택스(공동인증서) 또는
팝빌·바로빌 등 **발급대행 API** 연동이 필요합니다(서버·인증서·계약 필요).
이 페이지는 계산서 **양식 작성·인쇄/PDF·이메일 첨부**용이며, 추후 API 연동으로 정식 발행까지 확장할 수 있습니다.
