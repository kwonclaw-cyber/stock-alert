# 총겜동 내수서버 길드 운영 페이지

숲(SOOP) 방송인들이 운영하는 마인크래프트 **내수서버**의 길드 운영 웹페이지입니다.
온라인 라이브 공유를 전제로 하며, **최초 접속 시 공용 비밀번호 게이트**로 보호됩니다.

## 페이지(탭) 구성

1. **길드별 멤버현황** — 길드별 멤버 명단(이름·직업). 다른 페이지에 자동 연동
2. **박사장길드 내실현황판** — 메인 길드 스텟/잠재/방어구/장신구/탈것 + 곡괭이 1★~5★
3. **보스타이머** — 처치→젠 카운트다운, 보스별 소리·알림(🔔)
4. **광산타이머** — 광산 일괄 생성, 완료→쿨타임, 채굴가능/남은시간순 정렬, 지도 마커, 네비 1·2·3
5. **철타이머** — 문파원별 철넣기 완료/주기/일별 누적 카운트
6. **일숙** — 문파원×숙제 완료 체크(날짜별)
7. **무인의 거처 / 제작 및 재료 정보 / 마을지도** — 캡처 카드·지도 이미지 모음
8. **부적** — 부적 계산기 + 옵션 효과표
9. **이미지 합치기 도우미** — 여러 이미지를 가로/세로로 이어붙여 한 장으로
10. **히든추리용** — 단서 순서 정리 + 캡처 붙여넣기 + 결론
11. **정보공유** — 공략·꿀팁·공지 게시판

모든 데이터는 브라우저에서 바로 수정하면 서버에 자동 저장됩니다.

## 기술 스택

- **Next.js 15** (App Router) + React 19 + TypeScript + **Tailwind CSS v4**
- **jose** + Next.js 미들웨어 기반 공용 비밀번호 접속 게이트
- 저장: **Vercel KV(Upstash Redis)** 연결 시 실시간 공유 / 미연결 시 로컬 파일

## 보안 동작 방식

- 모든 페이지는 미들웨어(`middleware.ts`)에서 세션 쿠키를 검사합니다.
- `/login`에서 공용 비밀번호(`SITE_PASSWORD`)를 입력하면 서명된 JWT가
  `HttpOnly` · `Secure` 쿠키로 발급됩니다. (기본 유효기간 7일, 상수시간 비교)

## 로컬 실행

```bash
npm install
cp .env.example .env.local   # 값을 채워주세요
npm run dev
```

```bash
# .env.local
SITE_PASSWORD=원하는_공용_비밀번호
AUTH_SECRET=길고_랜덤한_시크릿   # 예: openssl rand -base64 32
```

## Vercel 배포 (라이브 공유)

### 1) 배포
1. GitHub 저장소를 Vercel에 **Import** 합니다. (Framework: Next.js 자동 인식)
2. **Settings → Environment Variables** 에 추가:
   - `SITE_PASSWORD` — 길드원에게 공유할 공용 비밀번호
   - `AUTH_SECRET` — 길고 랜덤한 시크릿 (외부 공유 금지)
3. **Deploy** → 발급된 URL을 길드원에게 공유.

### 2) 실시간 공유 켜기 (여러 명 동시 수정) — Vercel KV 연결
> 이 단계를 하지 않으면 입력값이 서버 인스턴스에 영구 저장되지 않습니다.
> (화면 우측 상단 배지가 **로컬 저장**으로 표시됩니다.)

1. Vercel 프로젝트 → **Storage** 탭 → **Create Database** → **Upstash for Redis(=KV)** 선택.
2. 생성한 DB를 **이 프로젝트에 Connect**. (그러면 `KV_REST_API_URL`, `KV_REST_API_TOKEN`
   환경변수가 자동으로 주입됩니다. `UPSTASH_REDIS_REST_*` 이름도 자동 지원됩니다.)
3. **Redeploy** 한 번 실행.
4. 접속 후 우측 상단 배지가 **공유 저장 ON** 으로 바뀌면 완료 — 모두 같은 데이터를 봅니다.

## 데이터 저장 방식

`lib/store.ts` 의 어댑터가 환경에 따라 자동 선택합니다.
- KV 환경변수 있음 → Upstash Redis(REST)에 단일 키로 저장 → **실시간 공유**
- 없음 → `.data/store.json` 로컬 파일(개발용 / 단일 인스턴스)
