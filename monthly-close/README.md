# 월마감 정산 페이지

브랜드별·매장별 **매출 / 전용물류매출 / 전용이익**을 전월대비로 산출하는 단일 HTML 페이지입니다.
서버 없이 브라우저에서 동작하며, 업로드한 데이터는 외부로 전송되지 않습니다.
기존에 수기로 쓰던 "월마감 엑셀(브랜드관리팀 SV월보)"의 계산식을 그대로 구현했습니다.

## 바로 쓰는 법 (가장 간단)

`월마감.html` **한 파일만 받아 더블클릭**하면 됩니다. 인터넷 연결·설치 불필요,
모든 처리가 브라우저 안에서만 이루어져 매출·매입가 등 데이터가 외부로 전송되지 않습니다.
(SheetJS 라이브러리와 매입가·매장 매칭표가 파일 안에 모두 내장돼 있습니다.)

> 개발/유지보수용 분리본은 `index.html` + `vendor/` + `data/` 폴더입니다.
> 내장 데이터(`data/*.js`)나 화면(`index.html`)을 수정한 뒤
> `python3 build_single.py` 를 실행하면 `월마감.html` 단일 파일이 다시 생성됩니다.

## 매월 업로드하는 파일

| 구분 | 내용 | 비고 |
|---|---|---|
| **POS 매출 (브랜드별)** | matetech "기간별 매출분석 매장별분리" 내보내기 | 밥구정·육식 등 여러 개 한 번에 선택 |
| **물류 발주원장 (누적)** | 푸드가이드 "상품별 실적조회" 누적 원장 | 발주일·브랜드·전/범용·공급가·총합계 포함 |
| 전용상품 매입가 (선택) | 신규 전용상품 매입가 보충용 | `매입가?` 경고가 뜰 때만 |
| 전월 결과파일 (선택) | 지난달 "결과 내보내기" 파일 | 매출 전월대비용 |

업로드 후 **대상월(당월)** 을 고르면(기본=물류원장 최신월) 자동 계산합니다.
전월(전월대비)은 물류원장의 직전월 데이터로 자동 비교하며, 매출 전월대비는 전월 결과파일이 있어야 표시됩니다.

## 계산식 (월마감 엑셀과 동일)

| 항목 | 정의 |
|---|---|
| 매출 | POS의 선택 컬럼 (기본 **순매출액**, VAT제외). 실매출액·매출액으로 변경 가능 |
| **전용물류매출** | 물류원장에서 `전/범용 = 전용` 행의 **총합계(공급가+부가세)** 합 (매장별) |
| **전용이익(본사이익)** | 전용 행의 **(공급가 − 매입가×수량 − 공급가×물류수수료율)** 합 |
| 물류수수료 | 공급가 × **8.2%** (페이지에서 변경 가능) |
| 매입가 | 내장 매입가 마스터(`data/buyprice.js`)에서 상품코드로 조회 |
| 전용발주율 | 전용물류매출 ÷ 매출 |
| 전용이익률 | 전용이익 ÷ 전용물류매출 |
| 전월대비(금액) | (당월 − 전월) ÷ 전월 (%) |
| 전월대비(비율) | 당월 − 전월 (%p) |

검증: 원본 월보(202604)와 매장별 본사이익이 정확히 일치합니다
(예: 광주서구본점 1,528,332 / 영등포 1,284,498).

## 사입매장 자동 분류

각 브랜드 **기준매장**의 전용발주율보다 임계값(기본 **7.5%p**) 이상 낮은 매장을 사입으로 분류합니다.

- 기본 기준매장: 밥구정로데오 = `광주서구본점`, 육식사관학교 = `남양읍점`
- 기준매장·임계값은 페이지에서 변경 가능(브라우저에 저장)

## 내장 참조 데이터 (`data/`)

- `buyprice.js` — 전용상품 매입가 마스터 (상품코드 → 매입가)
- `storematch.js` — 매장 매칭표 (메이트코드 ↔ 물류코드 ↔ 매장명 ↔ 브랜드)

> **신규 전용상품 매입가 (추후 추가 반영)**: 어떤 상품이 새로 "전용"으로 전환되면 매입가 마스터에 없을 수 있습니다.
> 이때 매장에 `매입가?` 배지와 상단 경고가 표시되며, 해당 전용이익이 과대 계산됩니다. 보충 방법:
> 1. **화면에서 직접 입력** — 경고에 누락 상품(코드·상품명·관련 공급가)이 표로 나오며, 각 행의 칸에 매입가를 입력하고 저장하면 즉시 반영됩니다.
> 2. **파일 업로드** — `상품코드·매입가` 컬럼이 든 파일을 "전용상품 매입가" 칸에 올리면 한 번에 보충됩니다.
>
> 추가한 매입가는 **브라우저에 영구 저장**되어 다음에 열 때도 자동 적용됩니다(같은 브라우저 기준).
> 모든 사용자/기기에 영구 반영하려면 `data/buyprice.js`(내장 마스터)를 갱신해 커밋하세요.

## 결과 내보내기

`결과 엑셀 내보내기`로 저장하면 브랜드·매장별 전 항목이 담긴 파일이 생성됩니다.
이 파일을 다음 달 "전월 결과파일"로 올리면 매출 전월대비가 표시됩니다.
