🧭 표본매장 프로젝트 — 진행 기록 & 로직 정리

처음 접근부터 시행착오·각 단계 결과·최종 결론까지 한 페이지로

0 · 목표

브랜드별 표본매장을 정하고, 그 매장의 배민/쿠팡 운영 변경(광고율·즉시할인·쿠폰·운영시간 등)이 매출 변동과 어떤 관계인지 분석할 수 있는 표본 데이터를 만든다.

1 · 요구사항 확정 (질문으로 좁힘)

질문결정
운영변경 데이터 소스배민/쿠팡 사장님 사이트에서 수집
표본매장 선정브랜드별 수기 지정
계정 구조매장별 개별계정
작업 성격일회성, 로그인은 사용자가 직접
핵심 데이터현재값이 아니라 “변경 이력”(시간축)
결과물매출과 결합한 표(엑셀/시트)
방향 전환 포인트: 처음엔 “현재 설정값”을 받으려 했지만, 실제로 필요한 건 “언제 무엇을 어떻게 바꿨나”였다. 그리고 배민에 정확히 그걸 주는 변경이력 화면이 있었다.

2 · 시도와 결과 (로직이 진화한 과정)

시도 1 화면 저장 방식 capture.py

로그인 후 변경이력 페이지의 HTML·스크린샷·네트워크 JSON을 통째로 저장.

결과: 성공. 96개 응답 캡처 → 99건 파싱.

핵심 발견: 배민 변경이력은 깔끔한 JSON API(modify-history/shop · instant-discount · ad-campaign)이고, 변경 전→후 값이 그대로 들어있다.

한계: 수동 클릭 의존, 페이지네이션이 중간에 잘려 6개월 완전판이 안 됨.

시도 2 자동수집기 baemin_history.py (여러 버전)

“로그인만 하면 6개월치를 자동으로” 목표로 Playwright 자동화 반복 시도.

방식결과
외부에서 API 호출(ctx.request)403 (인증 토큰 없음)
페이지 안 fetch(page.evaluate)CORS / 실행컨텍스트 파괴(-1)
루트 시작 + 로그인 가드로그인 창이 추적 밖으로 이탈
실제 Chrome(persistent)로그인된 창을 Playwright가 못 잡음

결론: Playwright로는 막힘. 배민 SSO 로그인이 자동화가 제어 못 하는 별도 창으로 빠진다.

이때 배운 진짜 로직: 자동화의 벽은 데이터가 아니라 “인증된 세션을 바깥에서 건드릴 수 없다”(SSO·2FA·봇차단)는 점이었다.

전환 · 정답 개발자도구 콘솔 스니펫 baemin_console.js

발상 전환: 우리가 요청을 만들지 말고, 로그인된 페이지가 스스로 부르는 응답을 가로채자.

  • 토큰 실어 replay → 여전히 토큰 못잡음·403·CORS
  • 최종 “응답 가로채기” → 토큰·CORS·403 문제가 원천적으로 사라짐

성공! 밥구정로데오(14323681) 6개월 변경이력 190건 수집 (가게 80 · 광고 84 · 즉시할인 26).

결합 매출 합치기 merge_sales.py

matetech 월별 엑셀(주문경로별=배민 일별매출, 시간대별)을 6개월 합본하고, 변경이력과 날짜로 조인.

결과: 일자별 타임라인 169일 + 시간대별 2,033행 + 변경이력 190건 통합 워크북.

3 · 최종 핵심 로직

  1. 운영변경은 배민, 매출은 matetech — 두 출처를 날짜로 잇는다.
  2. 배민 변경이력은 JSON API로 깔끔하게 있다(변경 전→후 포함).
  3. 자동화의 벽은 인증. 해법은 “로그인된 페이지 안에서, 앱이 만든 응답을 가로채기” — 토큰·CORS·차단을 모두 우회.
  4. 사람은 로그인 + 탭 조회만, 나머지(누적·정렬·표 변환)는 코드가.

4 · 확립된 프로세스

콘솔 코드 붙여넣기→ 탭별 조회 + __scrollAll()→ __save() → JSON→ matetech 월별 엑셀→ 합치기 → 통합 워크북

매장 한 곳당 위를 반복 → 매장별 표본 데이터 완성.

7 · 한계 & 다음 단계

한계
  • 시간대별 매출은 “배달” 합산(배민+쿠팡)이라 배민만 시간대별은 matetech에서 안 나옴.
  • 종료된 광고 캠페인 이력은 배민 API가 일부 제한.
  • 쿠팡이츠는 아직 미진행(변경이력 화면 확인 필요).
다음
  • 다른 표본매장으로 같은 프로세스 확대 → 여러 매장 합본.
  • 쿠팡이츠도 동일한 콘솔 방식 적용 검토.
  • “변경 전후 N일 매출 비교” 자동 요약 시트 추가 가능.
