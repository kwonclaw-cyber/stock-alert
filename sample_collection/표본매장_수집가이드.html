📊 표본매장 데이터 수집 가이드

배민 운영 변경이력 × 매출 — 매장별 표본 데이터 만드는 전체 과정

한눈에 보는 흐름

① 배민 변경이력 수집
(콘솔 코드) → ② matetech 매출 받기
(월별 엑셀) → ③ 파일 전달 → ④ 통합 워크북
(매출×변경)

매장 한 곳당 ①·②를 하고 파일을 넘기면, 일자별·시간대별로 매출과 운영변경이 합쳐진 엑셀이 만들어집니다.

STEP 1 · 배민 변경이력 수집 (개발자도구 콘솔)

배민 사장님사이트가 스스로 불러오는 데이터를 가로채 모으는 방식이라 설치·로그인 자동화가 전혀 필요 없고, 토큰/차단 문제도 없습니다.

1
로그인
크롬에서 self.baemin.com 로그인 → 왼쪽 메뉴 변경이력 화면 열기
2
콘솔 열기
F12 → 상단 Console 탭. 아래 코드를 먼저 붙여넣고 Enter.
※ 붙여넣기가 막히면 콘솔에 allow pasting 입력 후 다시 붙여넣기
3
가게 탭
가게 탭 클릭 → 파란 조회 → 콘솔에 +10 가게 … 찍히면 성공 → 콘솔에 __scrollAll() 입력하면 끝까지 자동으로 다 불러옵니다
4
즉시할인 탭
즉시할인 탭 → 조회 → __scrollAll()
5
광고·서비스 탭
광고·서비스 탭 → (캠페인/변경항목 골라가며) 조회 → __scrollAll() (캠페인 여러 개면 바꿔가며 반복)
6
저장
콘솔에 __save() 입력 → baemin_변경이력_매장번호.json 파일 다운로드 ✅
핵심 순서: 반드시 코드 붙여넣기 → 그다음 조회. 조회를 먼저 하면 데이터를 못 잡습니다.
여러 매장: 가게 드롭다운만 바꿔 조회 → __scrollAll() → __save() 하면 매장마다 파일이 따로 나옵니다. 다른 계정이면 그 계정으로 로그인 후 동일하게.

📋 붙여넣을 콘솔 코드 (복사 버튼 사용)


  


  
  
STEP 2 · matetech 매출 받기 (월별 엑셀)

  
matetech(메이트포스)는 1개월씩만 받을 수 있어, 6개월이면 6개씩 나눠 받습니다. 두 종류를 받으세요.

  

    
① 주문경로별 매출분석 — 배민/쿠팡 분리용

    
1
matetech 로그인 → 매출분석 → 주문경로별 매출분석

    
2
기간을 한 달씩 지정 (예: 2025-12-01 ~ 2025-12-31)

    
3
조회 → 엑셀 다운로드. 6개월이면 월별로 6번 반복

    
② 시간대별 매출분석 — 시간대 분석용

    
1
매출분석 → 시간대별 매출분석에서 동일하게 월별로 받기

  

  
참고: 시간대별은 채널이 내점/배달/포장으로만 나뉘어 “배달” = 배민+쿠팡 합산입니다(배민만 따로는 안 나옴). 운영시간 분석엔 충분합니다.


  
  
STEP 3 · 파일 전달

  

    
매장 한 곳당 모인 파일을 넘기면 됩니다:

    
    
      
  • baemin_변경이력_매장번호.json (STEP 1 결과) — 1개
    • 
      
  • 주문경로별 엑셀 — 월별 (6개월이면 6개)
    • 
      
  • 시간대별 엑셀 — 월별 (6개월이면 6개)
    • 
    

    
한 번에 5개까지 업로드되니, 많으면 zip으로 묶어서 전달하면 됩니다.

  


  
  
STEP 4 · 결과물 (자동 생성)

  
전달한 파일로 아래 3개 시트짜리 통합 워크북이 만들어집니다.

  

    
시트내용

    
일자별 타임라인날짜·요일·배민매출·건수·쿠팡매출·총매출 + 그날 운영변경 (변경일은 노란색 강조)

    
시간대별날짜·시간대·배달건수·배달매출 + 운영시간 변경 표시

    
배민 변경이력운영시간·즉시할인·광고 입찰/예산 변경 원본 (변경 전→후)

  

  
이렇게 하면 “광고 입찰을 올린 날·즉시할인을 켠 날·운영시간을 늘린 날” 전후로 매출이 어떻게 움직였는지 한눈에 비교됩니다.
