육식사관학교 전 매장의 배민 운영 변경(광고·즉시할인·영업시간)이 배민 매출에 어떤 영향을 주는지, 6개월 일별 데이터로 분석한다.

1 · 핵심 전환 — 매출 수집 자동화

처음 수동 다운로드의 벽

메이트포스에서 매장×날짜 매출을 받으려면 주문경로 182번 + 시간대 182번 수동 다운로드가 필요했다. 날짜를 매번 바꿔 클릭 = 사실상 불가능.

정답 API 자동수집 matetech_console.js

검색 버튼이 부르는 orders-path API를 찾아, 날짜를 자동 루프로 호출. 로그인된 세션에서 콘솔에 붙여넣고 1번 실행 → 6개월 전체가 파일 하나로.

검증: 수집 합계 = 공식 일별 총매출과 정확히 일치(이중계상·누락 없음).

결합 매장 매핑 + 마스터 build_master.py

메이트 매장코드 ↔ 배민 매장번호를 매장명으로 매핑(185개 연결), 매출과 변경이력을 날짜로 조인.
