    📊 분석 보드 — 변경이 매출에 준 직접 영향

    각 카드의 큰 숫자 = 순효과(%p) = 그날 매출(전주 동일요일 대비) − 기준선. 변경만의 영향에 가깝게 보정한 값입니다. ＋면 매출 견인, －면 매출 하락.

    해석 주의: "변경한 날 하루" 비교는 사장님 반응(매출 부진 → 광고·할인 ON)이 섞여 역인과일 수 있습니다. 지속적 변경(운영시간 등)과 주차·월 비교 시트를 함께 보세요.

    월별 비교

    ⚠️ 이 페이지는 엑셀 처리를 위해 인터넷에서 라이브러리(SheetJS)를 한 번 불러옵니다. 인터넷이 연결돼 있어야 합니다.

    📌 프로세스 가이드

    ① 배민 변경이력(콘솔)→ ② matetech 매출 엑셀→ ③ 위 박스에 끌어다 놓기→ ④ 분석 다운로드

    STEP 1 · 배민 변경이력 받기 (콘솔)

    1
    크롬에서 self.baemin.com 로그인 → 왼쪽 변경이력 화면
    2
    F12 → Console 탭. 아래 코드를 먼저 붙여넣고 Enter (막히면 allow pasting 입력)
    3
    가게 탭 → 조회 → __scrollAll()  /  즉시할인 → 조회 → __scrollAll()  /  광고·서비스 → (캠페인 선택) 조회 → __scrollAll()
    4
    콘솔에 __save() → baemin_변경이력_매장번호.json 다운로드
    

    
    STEP 2 · matetech 매출 받기 (월별)
    
    
    1
    매출분석 → 주문경로별(배민/쿠팡 분리)·시간대별을 월 단위로 조회 → 엑셀 다운로드 (6개월이면 각 6개)
    
    
    2
    주문경로별이 없으면 시간대별만 있어도 됩니다(매출은 '배달' 합산으로 표시).
    

    
    STEP 3 · 이 페이지에 올리기
    
    
    1
    받은 JSON + 매출 엑셀을 맨 위 박스에 끌어다 놓고 분석 만들기 ▶ → 분석.xlsx / 통합본.xlsx 다운로드
