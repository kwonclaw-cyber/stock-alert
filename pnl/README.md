# 가맹점 손익계산서(P&L) 생성기

배달전문 음식점 손익모델로 매장의 손익계산서를 만들고, 개선점을 도출해
**이미지로 저장·복사**해서 매장에 전달하는 단일 HTML 페이지입니다.
서버 없이 브라우저에서 동작하며, 입력 데이터는 외부로 전송되지 않습니다.

## 바로 쓰는 법

`손익계산서.html` **한 파일만 받아 더블클릭**하면 됩니다. (인터넷·설치 불필요, 이미지 캡처 라이브러리 내장)

1. **매장명**·기준월·주문건수(선택)를 입력합니다.
2. 매출/변동비/고정비 항목별 금액을 입력하면 손익계산서가 실시간으로 생성됩니다.
3. 오른쪽 리포트 상단의 **이미지 저장(PNG)** 또는 **이미지 복사**로 받아 매장에 전달합니다.

> 개발/유지보수용 분리본은 `index.html` + `vendor/html2canvas.min.js` 입니다.
> 수정 후 `python3 build_single.py` 를 실행하면 `손익계산서.html` 단일 파일이 다시 생성됩니다.

## 손익 모델 (배달전문 음식점)

- **매출**: 배달 / 포장 / 홀 매출 → 총매출
- **변동비**(매출 연동): 식자재비, 배달대행비, 플랫폼 중개수수료, 결제·PG 수수료, 포장재·소모품비, 광고선전비
- **공헌이익** = 총매출 − 변동비
- **고정비**(판매관리비): 인건비, 임차료, 관리비, 수도광열비, 통신비, 로열티·가맹수수료, 보험·세금, 기타
- **영업이익** = 공헌이익 − 고정비, **영업이익률** = 영업이익 ÷ 총매출

각 항목은 매출 대비 비중(%)으로 표시됩니다.

## 개선점 분석

매출 대비 비중이 권장치를 초과한 항목을 자동으로 찾아 개선 제안을 카드로 보여줍니다
(예: 식자재비 35% 초과 → 레시피 표준화·로스 관리). 영업이익률은 적자/낮음/양호/우수로 평가합니다.
권장치는 업종·지역에 따라 달라질 수 있는 일반 기준값이며, `index.html` 의 `VARIABLE`/`FIXED` 정의에서 조정할 수 있습니다.
