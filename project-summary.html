🔬 백테스트 엔진 구조

기술 지표

순수 TypeScript 구현

  • EMA / SMA (지수·단순 이동평균)
  • RSI — Wilder 평활화
  • MACD (fast/slow/signal)
  • 볼린저 밴드
  • ATR (Average True Range)
  • VWAP
성과 지표

10가지 메트릭

  • 총수익률 / 연환산수익률
  • 샤프 비율 (252일 기준)
  • 소르티노 비율 (하방 전용)
  • 칼마 비율 (연수익 ÷ 최대낙폭)
  • 최대 낙폭 (에쿼티 커브 기준)
  • 승률 / Profit Factor / 기댓값
  • 평균 보유일수
데이터 소스

3개 데이터 피드

  • Yahoo Finance — 미국주식·지수·외환
  • Binance API — 암호화폐
  • 토스증권 Open API — 한국주식
진입/청산 로직

룩어헤드 바이어스 없음

  • 진입 조건 (AND 방식, 최대 5개)
  • 필터 조건 (큰 그림 추세 필터)
  • 청산 조건
  • SL — bar low 기준 즉시 청산
  • TP1 (50% 청산) / TP2 / 트레일링
  • 포지션 크기: 리스크% or 고정%

🚀 Vercel 배포 방법 (5분)

1

GitHub에 push

현재 브랜치 claude/lucid-hamilton-hmvg60를 main에 머지하거나 직접 배포

2

vercel.com 접속 → New Project

GitHub 저장소 kwonclaw-cyber/stock-alert 선택 · Root Directory: . (기본값)

3

Environment Variables 입력

위 환경 변수 4개 입력 (SITE_PASSWORD, AUTH_SECRET, TOSS_INVEST_API_KEY, ANTHROPIC_API_KEY)

4

Deploy 클릭

약 1~2분 후 your-project.vercel.app URL 발급 · 이후 push 시 자동 재배포

5

(선택) Vercel KV 연결

Storage 탭 → Upstash Redis 생성 → 프로젝트 연결 → 여러 명이 실시간 데이터 공유 가능
